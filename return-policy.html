Return Policy

Last Updated: June 2025

Important Notice: As an Amazon affiliate, NEUCART does not handle returns directly. All return requests must be processed through Amazon's return system.

1. Amazon's Return Policy

All products listed on NEUCART are sold and fulfilled by Amazon. Amazon's return policy applies to all purchases made through our affiliate links.

Key points about Amazon's return policy:

  • Most items can be returned within 30 days of receipt
  • Some products have different return policies (electronics, luxury items, etc.)
  • Amazon provides prepaid return shipping in many cases

View Amazon's complete return policy: Amazon.in Returns Center

2. How to Initiate a Return

To return an item purchased through NEUCART:

  1. Go to Amazon's Returns Center
  2. Select the order containing the item you want to return
  3. Choose a return reason and follow the instructions

3. Refund Processing

Refunds are issued by Amazon according to their policies:

  • Refund method depends on your original payment method
  • Processing time varies by payment method (3-10 business days typically)
  • Amazon will notify you when your refund is processed

4. Non-Returnable Items

Certain items cannot be returned through Amazon's policy, including:

  • Downloadable software products
  • Gift cards
  • Some health and personal care items

Check the product page on Amazon for specific return eligibility.

5. Our Role

As an Amazon affiliate, NEUCART:

  • Cannot process returns directly
  • Cannot expedite return processing
  • Cannot override Amazon's return decisions

For any return-related issues, please contact Amazon customer service directly.
